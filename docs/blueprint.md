# **App Name**: LoanEase

## Core Features:

- Loan Input: Input loan details: principal, interest rate, and loan duration.
- Interest Type Selection: Select between compound and simple interest calculations.
- Amortization Schedule Generation: Generate a detailed amortization schedule showing each payment.
- Payment Schedule Display: Display the payment schedule in a clear table.
- Payment Summary: Summarize total payments, interest paid, and principal paid.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trustworthiness and finance.
- Background color: Light gray (#ECEFF1), desaturated to 20%, for a clean, professional look.
- Accent color: Slightly lighter blue (#5C6BC0), differing by 15 in brightness and saturation from the primary blue.
- Body and headline font: 'Inter', a sans-serif with a modern, machined, objective, neutral look, suitable for both headlines and body text.
- Use simple, geometric icons for financial terms and functions.
- Use a tabular layout to display amortization schedules.
- Subtle animations on data updates and interactions.
