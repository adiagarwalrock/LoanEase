# LoanEase

A clean, fast loan calculator built for students to understand amortization schedules and payment breakdowns.

## Features

*   **Loan Calculator**: Calculate monthly payments based on principal, rate, and term.
*   **Flexible Options**: Support for Simple and Compound interest.
*   **Detailed Schedule**: View monthly or yearly amortization breakdowns.
*   **Export**: Print or save your schedule as PDF.
*   **Private**: All calculations happen in your browser; no data is stored.

## Tech Stack

*   **Frontend**: [Vite](https://vitejs.dev/) + [Vue 3](https://vuejs.org/) + [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Components**: [shadcn-vue](https://www.shadcn-vue.com/)

## Getting Started

1.  **Clone & Install**
    ```bash
    git clone https://github.com/adiagarwalrock/LoanEase.git
    cd LoanEase
    npm install
    ```

2.  **Run Locally**
    ```bash
    npm run dev
    ```

3.  **Build**
    ```bash
    npm run build
    ```
